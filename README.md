# grey-harbor
Grey Harbor Intelligence Training Simulator
Grey Harbor is a browser-based training simulator designed to develop critical intelligence analysis skills in military and national security professionals. Players work through 28 sequential intelligence items drawn from a realistic contested coastal scenario, making triage decisions on reports from HUMINT, SIGINT, IMINT, OSINT, and CYBER sources. For each item, the trainee selects one of six actions — Elevate to Commander, Correlate, Flag Ambiguous, Defer, Dismiss, or Request Collection — and receives immediate feedback from three AI advisory agents (Screening, Fusion, and Attention) as well as an after-action coach. The scenario is designed to surface common cognitive traps including automation bias, noise tolerance, and premature escalation.

The simulator runs entirely in the browser with no installation required and works on both desktop and mobile devices. Two versions are provided: index.html is a responsive single-file application that automatically displays a three-column interface on desktop and a streamlined single-column interface on mobile. Grey_Harbor_Simulator_v2.html is a desktop-optimized version with optional live AI agent responses via the Anthropic API. Both versions run in pre-baked mode by default, requiring no API key or internet connection beyond loading the page. The simulator was developed at Carnegie Mellon University's Heinz College by David Steier as part of research on AI-augmented critical thinking support in high-stakes analytical environments.
